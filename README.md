# OCP2_CV_web
Projet dans le cadre de la formation OC Web Dev : "Vous venez de vous lancer en tant que développeur web freelance et un client potentiel, qui gère une entreprise de toilettage pour chien, vous repère sur LinkedIn. Cependant, votre CV n'est pas tout à fait à jour. Pour pouvoir répondre à ses questions (et trouver de nouveaux clients) vous décidez de créer votre propre site web pour présenter votre CV et ainsi être capable de répondre à ses questions. Vous choisissez de développer votre site en HTML et CSS."
